# 📅 내 달력 — 기능 정리

> `calendar.html` 한 파일로 동작하는 인터랙티브 달력 웹 페이지입니다.

---

## 1. 화면 구성

| 영역 | 위치 | 설명 |
|------|------|------|
| **상단 바 (Top Bar)** | 화면 최상단 | 제목, 오늘 날짜, "오늘로 이동" 버튼 |
| **달력 (Calendar)** | 왼쪽 (넓게) | 월별 날짜 그리드, 월 이동 버튼 포함 |
| **일정 패널 (Event Panel)** | 오른쪽 | 선택한 날짜의 일정 목록 + 일정 추가 폼 |

---

## 2. 달력 탐색

- **월 이동** — 달력 상단의 `‹` / `›` 버튼으로 이전·다음 달 이동
- **오늘로 이동** — 상단 바의 버튼 클릭 시 현재 월로 돌아가고 오늘 날짜 자동 선택
- **오늘 날짜 강조** — 오늘 날짜는 파란색 원형 배경으로 항상 강조 표시
- **선택된 날짜** — 클릭한 날짜는 파란 테두리로 하이라이트

---

## 3. 일정 추가

날짜를 클릭하면 오른쪽 패널 하단에 **일정 추가 폼**이 나타납니다.

### 입력 항목

| 항목 | 필수 여부 | 설명 |
|------|----------|------|
| **색상 선택** | 선택 | 4가지 색상(파랑·분홍·초록·노랑) 중 하나 선택 |
| **시간** | 선택 | `HH:MM` 형식의 시간 선택기 (비워두면 시간 없음) |
| **일정 내용** | 필수 | 최대 40자 텍스트 입력 |

### 추가 방법

- `＋ 추가하기` 버튼 클릭
- 또는 텍스트 입력 후 `Enter` 키

---

## 4. 일정 표시

### 달력 셀 내 표시
- 해당 날짜 칸에 일정이 **색상 태그**로 표시됨
- 최대 **3개**까지 표시 (초과분은 미표시)
- 시간이 있는 경우 `08:56 수학 시험` 형태로 표시
- **시간 오름차순 정렬** (빠른 시간 → 늦은 시간 순, 시간 없는 항목은 맨 아래)

### 오른쪽 패널 목록
- 선택한 날짜의 **모든 일정**을 목록으로 표시
- 시간이 있는 경우 왼쪽에 **파란 배지**로 시간 표시
- 시간 오름차순으로 자동 정렬

---

## 5. 일정 삭제

- 일정 항목 오른쪽의 `×` 버튼 클릭으로 즉시 삭제

---

## 6. 색상 시스템

| 번호 | 색상 | 용도 예시 |
|------|------|----------|
| 1 | 🔵 파랑 (기본) | 일반 일정 |
| 2 | 🩷 분홍 | 개인 일정 |
| 3 | 🟢 초록 | 학교·공부 |
| 4 | 🟡 노랑 | 중요 메모 |

---

## 7. 데이터 저장

- 모든 일정은 **브라우저 localStorage**에 자동 저장
- 페이지를 닫거나 새로고침해도 데이터 유지
- 별도 서버·계정 불필요
- 저장 키: `calEvents2`

> ⚠️ **주의**: 브라우저의 캐시/사이트 데이터를 삭제하면 일정도 함께 삭제됩니다.

---

## 8. 디자인 특징

- **하늘색 + 흰색** 혼합의 밝고 청량한 테마
- **글래스모피즘** — 반투명 카드 UI
- **2컬럼 레이아웃** — 스크롤 없이 한 화면에 모든 기능 표시
- **애니메이션** — 배경 오브 부유 효과, 카드 호버 시 확대 효과
- **반응형** — 화면이 좁아지면 1컬럼으로 자동 전환

---

## 9. 사용 방법 요약

```
1. calendar.html 파일을 브라우저에서 열기
2. 달력에서 날짜 클릭
3. (선택) 시간 선택기에서 시간 입력
4. 일정 내용 입력 후 Enter 또는 추가하기 버튼 클릭
5. 일정 삭제 시 목록의 × 버튼 클릭
```
